docker_python note

----------------------------------------------------------------

I did finally manage to get "debugging Python in Docker-compose with bind-volumes in vscode" working, but not through vscode's "launch-Docker-compose" concept.
Basically I had to "insert" debugpy in the code, and run docker-compose in terminal, and then attach vscode to the running container.
I found a recipe for this here:
https://www.python-engineer.com/posts/debug-python-docker/

I would love to see how/if this can be achieved more conveniently through vscode ... when vscode is ready ...
